# Sorting Techniques

## Types of Sorting Techniques

- Bubble Sort
- Insertion Sort
- Selection Sort
- Heap Sort
- Merge Sort
- Quick Sort
- Tree Sort
- Shell Sort
- Count Sort
- Bucket/Bin Sort
- Radix Sort

## Criteria for Analysis:

1. Number of comparisons.
2. Number of swaps.
3. Adaptive.
4. Stable.
5. Extra Memory.
